# AI-Assisted Smart Thermal-Controlled Dual Battery Switching System for EVs

## Overview

This project presents an intelligent dual-battery thermal management system designed for electric vehicles operating under continuous high-load conditions.

The system uses TinyML-assisted decision-making and real-time thermal monitoring to automatically switch between two battery packs based on temperature conditions.

When the active battery reaches a predefined thermal threshold:
- Cooling mechanism activates
- Battery switching occurs
- Secondary battery powers the vehicle
- Overheated battery cools down

This improves:
- Battery safety
- Thermal reliability
- Continuous vehicle operation
- Battery lifespan

---

## System Workflow

```text
Battery 1 Active
        ↓
Temperature Threshold Reached
        ↓
Cooling Pump ON
        ↓
Switch to Battery 2
        ↓
Battery 1 Cool Down
        ↓
Repeat Process
```
